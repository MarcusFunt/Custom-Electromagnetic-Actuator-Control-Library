Start a search with the run_optimization MCP tool, then open its build/optimize_results/latest.json here. In Chrome/Edge, "Open results file…" keeps watching the file and updates this page live as new generations complete; other browsers can re-open or drop the file again to refresh.

Auto-detects three JSON shapes: an optimizer status/result snapshot (has a "status" field), a simulate_design_detailed time series (has "t"/"x"/"v" arrays), and a sensitivity_sweep main-effect curve (has "knob"/"points") or a pairwise interaction grid (has "knob_a"/"knob_b"/"grid") from design_sensitivity.py. Nothing here is uploaded anywhere -- everything runs and stays in this page.
